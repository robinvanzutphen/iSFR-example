## iSFR example code

This repository contains example code accompanying the Scientific Reports publication:

**“Quantitative Model for Imaging Single Fiber Reflectance Spectroscopy”**

The code is intended solely as an illustrative example of the implementation described in the manuscript. It is provided to help readers understand the computational workflow and reproduce the core ideas presented in the paper.

### Tested environment

This example code was tested with:

- **MCX v2025 (“Jumbo Jolt”)**
- **MCXLAB**
- **MATLAB R2025a**
- **Ubuntu 24.04.2 LTS**

### Important note

This repository is **not intended to be actively maintained** to remain compatible with future software updates, package changes, or new MATLAB/MCX releases. It serves as an **example implementation only**, corresponding to the software environment listed above.

Users may therefore need to make minor adjustments when running the code in other environments or with newer software versions.
